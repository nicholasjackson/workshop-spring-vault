# Adding secrets to Vault

Let's now see how you can add a secret to Vault, a secret is simply a collection
of keys and values that are stored at a specific path.

You can find the commands for reading and writing secrets at the following location:

https://developer.hashicorp.com/vault/docs/secrets/kv/kv-v2#writing-reading-arbitrary-data

### Given the application name is `SpringVaultApplication` create a secret that has the key `password`

If you need some help, you can find the solution below.

<p>
<details>
  <summary style=\{{ fontSize: '18px', color: 'rgb(84, 128, 178)', fontWeight: 'medium' }} >Solution: configure secret</summary>

```
vault kv put secret/SpringVaultApplication password=H@rdT0Gu3ss
```

</details>
</p>
